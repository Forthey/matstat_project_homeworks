### Про автоматические ядерные оценки

$$

\xi = \frac{h}{b} \\

f(x) = \frac{1}{b}f_0(\frac{x-a}{b}), (b > 0, -\inf < a < \inf)\\

h_{opt} = b * \xi_{opt}

$$

### Проекционная оценка плотности вероятности

Суть проекционной оценки - мы берем некоторую ортонормированную систему функций $ \phi_i(x) $, причем без веса 

Тогда теоречический закон можно представить в виде разложения по этим функциям $ f(x) = \sum_{i=0}^{\inf} c_i\phi_i(x) $

Далее нужно смотреть размышления Ченцова. Основная идея Ченцова - $ \bar{c_i} $ можно взять как среднее арифметическое от системы функций, и затем оценка теоретического закона считается через разложение с оценками $ \bar{c_i} $, описанное выше

Варианты:

1. Случайная величина X распределена на полуоси x > 0. В этом случае в качестве $ \phi_i(x) $ удобно принять функции Логерра
2. Случайная величина X распределена на всей оси. В этом случае мы испоьзуем функции Эрмита. Также с этим некоторым образом связан полином Эрмита, для которого есть в том числе явное выражение.
3. Случайная величина X распределена в конечном интервале $ a \le x \le b $. Тогда лучше взять тригонометрическую систему функций

#### Задача - для своего закона получить проекционную оценку плотности вероятности

### Оценка методом максимального правдоподобия

Пусть имеется два параметра, $ f(x; a_1; a_2) $. Как мы строим оценку? Сначала оцениваем параметры, а затем в качестве оценки плотности берем параметризованную плотность с подставленными оценками параметров

Далее находим нормированную ОИСКО, используя оценку плотности
