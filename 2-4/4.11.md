# Условие задачи

Теоретический закон распределения выборки $ x_1, ..., x_n $ является законом Вейбулла-Гнеденко:

$$
f(x) = \lambda * c * x^{c - 1} * e^{-\lambda * x ^ c}, \lambda > 0, c > 0, x >= 0
$$

При $ \lambda = 1, c = 3, n = 400 $ построить ядерную оценку

$ f(x) = \frac{1}{nh} \sum_{i=1}^{n} K(\frac{x - x_i}{h})  $ (над f(x) волнистая линия)

С ядром В.А. Епанечникова

$$
K(x) =
\begin{cases}
\frac{3}{4\sqrt{5}}(1 - \frac{x^2}{5}), & \text{если } |x| <= \sqrt(5) \\
0, & \text{если } |x| >= \sqrt(5) \\
\end{cases}
$$

Вычислить ОИСКО $ \bar{\delta_n} $ и построить график этой функции в интервале $ (0, \inf) $
