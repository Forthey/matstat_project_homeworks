# Условие задачи

Теоретический закон распределения выборки $ x_1, ..., x_n $ является законом Вейбулла-Гнеденко:

$$
f(x) = \lambda * c * x^{c - 1} * e^{-\lambda * x ^ c}, \lambda > 0, c > 0, x >= 0
$$

Положим $ \lambda = 1, c = 3, n = 400 $ и построим по выборке проекционную оценку

$ f(x) = \sum_{i=0}^{N} c_i * \phi_i(x) $ (над f(x) и \phi_i(x) волнистая линия)

где $ \phi_i(x) $ обозначает функцию Лагерра

$ \phi_i(x) = e^{-x/2} \sum_{j=0}^{i} (-1)^{j}*1/(j!)C_{i}^{j}x^{j}, (i = [0; \inf]) $

Построить график зависимости ОИСКО $ \bar{\delta_{n}} $ в фукнции от N = 10 (10) 100
